SFTP & FTP

SFTP-administration

I din nuvarande prenumeration har du inte tillgång till SSH, men du kan använda SFTP för att göra en krypterad anslutning till ditt webbutrymme och överföra filer säkert. Uppgradera till ett av våra övriga paket om du vill använda SSH.

För att logga in med SFTP ska du använda:

ssh.affluere.se

affluere.se

22

Klicka på Skicka så får du ett e-postmeddelande som kan användas för att skapa/ändra SFTP-lösenord.

FTP-administration

Vi rekommenderar att du använder SFTP och inte FTP för att ansluta till ditt webbutrymme. SFTP är krypterat och därför säkrare att använda.

För att logga in med FTP ska du använda:

ftp.affluere.se

affluere.se

21

Klicka på Skicka så får du ett e-postmeddelande som kan användas för att skapa/ändra FTP-lösenord.
